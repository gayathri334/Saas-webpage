# 🌐 Responsive Web Application

A modern and responsive web application built using **HTML, CSS, JavaScript, and Node.js**. This project includes user authentication pages, a contact page, and a clean user interface designed for a seamless user experience.

## 🚀 Features

* 🔐 User Login & Signup System
* 🏠 Responsive Home Page
* 📞 Contact Page
* 🎨 Modern UI Design
* ⚡ Fast and Lightweight
* 📱 Mobile-Friendly Layout
* 🌍 Cross-Browser Compatibility
* 🖥️ Node.js Backend Integration

## 📂 Project Structure

```bash
├── README.md
├── index.html          # Main landing page
├── login.html          # User login page
├── signup.html         # User registration page
├── contact.html        # Contact page
├── styles.css          # Global styling
├── server.js           # Backend server
```

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* Node.js

## 📸 Project Overview

This project demonstrates the development of a complete front-end interface combined with a Node.js backend server. It provides a foundation for authentication-based web applications and can be extended with databases, APIs, and additional features.

## 🎯 Learning Outcomes

* Building responsive web pages
* Creating authentication interfaces
* Structuring web application files
* Styling with modern CSS
* Running a Node.js server
* Improving user experience and accessibility
